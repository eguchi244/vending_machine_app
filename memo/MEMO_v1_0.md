# 自販機プログラム
## 機能
- お金を受け取る
- 商品一覧を表示する
- お釣りを計算して返却する

## 非機能
- お金が投入されてない場合は？
- 存在しない商品が選択された場合は？
- お金が足りない場合は？

## 詳細な流れ
- お金を受け取る
- 商品一覧を表示する
- 商品を選択させる
- お釣りを計算する
- 購入品とお釣りの返却金額を表示する

## 課題
- UIが実装できていない -> v1_1で実装
- 下記の内容が加味されていない -> v1_1で実装
    - お金を受け取る→お金を各コインと紙幣で受け取る
    - お釣りを計算して返却する→各コインと紙幣で返却する
    - 貨幣に応じた下限と上限の設定
    - 連続購入できない
- 下記の内容が加味されていない -> v2_0以降で実装
    - 現金以外の決済方法（ICカード、クレジットカードなど）
    - お金投入前の選択式（ICカード、クレジットカード対応の機器は選択式）
    - 在庫管理（増減、上限、足りない場合）など
