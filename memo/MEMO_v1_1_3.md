# 自販機プログラム
## 機能
- サイドバー
    - お金を受け取る→お金を各コインと紙幣で受け取る New!
        - 貨幣に応じた下限と上限の設定 New!
    - メッセージパネルに各メッセージを表示する
    - お金の返却口でお釣りを返却する -> (各コインと紙幣で返却する) New!  -> v2_0以降で実装
- メイン画面
    - 自販機のタイトル
    - 商品一覧を表示する(多段表示)
    - 購入ボタンで決定する
    - お釣りを計算する

## 非機能
- お金が投入されてない場合は？
- 存在しない商品が選択された場合は？ -> ボタンならば発生しない
- 商品が選択されていない場合は？ -> ボタンならば発生しない
- お金が足りない場合は？

## 詳細な流れ
- 商品一覧の設定
- サイドバーUIを設定
    - お金を受け取る
    - 購入成功時以外のメッセージを表示
- メインUIを設定
    - 自販機のタイトルを表示
    - 商品一覧を表示する(多段表示)
    - 購入ボタンで決定する
        - お金が投入されてない場合 -> エラーメッセージ
        - それ以外（正常ケース）
        - お釣りを計算する
            - お金が足りている場合 -> 購入品とお釣りの返却金額を表示する(購入成功時)
            - お金が足りない場合 -> エラーメッセージ
